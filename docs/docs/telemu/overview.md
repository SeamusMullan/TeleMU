# TeleMU — Desktop Frontend

TeleMU is the Electrobun desktop application for viewing parsed telemetry data from Le Mans Ultimate.

## Stack

- [Electrobun](https://electrobun.dev/) — lightweight desktop app framework
- TypeScript
- Vite
- Tailwind CSS

## Project Layout

```
TeleMU/
├── src/                    # Application source
├── electrobun.config.ts    # Electrobun configuration
├── vite.config.ts          # Vite build config
├── tailwind.config.js      # Tailwind setup
├── tsconfig.json           # TypeScript config
└── package.json            # Dependencies (bun)
```
